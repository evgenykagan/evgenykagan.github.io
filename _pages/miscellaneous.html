UTD Rankings and a Possible Extension

The UTD Top 100 Rankings are a very useful public good. I started this side project as a respectful extension of the UTD framework, rather than an alternative to it (and also as a way to explore what can be done with the help of frontier agentic coding models ðŸ¤“ ). I made the following additions:

  1. OM Departments differ in size (and composition). Size adjustment can be difficult to do directly in the UTD interface. I added productivity adjustment based on OM faculty count.
  2. Counting. UTD counts each paper once for each school listed in the affiliations. I included an individual faculty-level view so that multiple same-school coauthors receive multiple credits.
  3. Citation adjustment. I also included a citation adjustment to add another view of impact.
  4. Multidisciplinary journals. Some journals are inherently multidisciplinary (looking at you, Management Science). Papers need to be classified as OM/OR vs. non-OM/OR. I use a combination of Management Science department labels and GenAI-based predictions for this.

This is only a first step: the current version covers 50 schools, U.S. only, and three operations journals (MS, OR and MSOM). Any comments are welcome!
